实习经历

小米公司 （ 2017年8月 ~ 2017年11月 ）

Ldap 专有账号服务器及管理系统 ( Ldap 协议部分实现+ SSM)

由于公司的部分下游系统无法接入单点登录，需要单独认证，为了规避账号泄露风险，需要专门开发一个ldap中间服务器，用以隔离用户与其存储在公司ldap服务器的密码，转而使用期存储在认证服务器中的账号密码。
在此项目中，我参照 RFC 2251 ，借助 Apache Ldap api，实现了一个非全功能的 Ldap 服务器，并使用ssm框架实现了对应的员工专有账号管理系统。目前系统已经接入公司的gitlab系统，之后可能也会接入其他系统。

公司商事合规管理系统 （SSM+G2+前端）

这是我在小米实习时做的第二个项目，此系统是为法务部门开发，用于处理投诉，举报，案件，以及统计相关数据。我在原有项目基础上添加了关键字查询，备注栏，数据图表统计等功能。前端借助G2框架实现，由于框架未提供项目需要的较为复杂格式的表格功能，我手写了 js 解析 json 生成复杂表格的代码。目前项目已经投入使用，受到领导好评。

外包项目

为了横向扩展知识面，接触不同领域的开发技术，扩展视野，我开了一个淘宝店，实现了一些小型外包项目，下面是部分项目简述。

  • 股票走势图项目 读取Excel表格数据，根据用户指定的时间等信息绘制股票蜡烛图。用python实现，使用 turtle 框架绘图。图形可以随数据量大小而自动适应调整。

  • 操作系统探测项目 根据不同操作系统对 TCP 数据包（FIN包）的不同反应（指纹），来推测指定ip的主机的操作系统。用 C++ 实现，主要原理为手动封装RAW_SOCKET。
  • IP包流量监控项目 借助Jpcap来对IP包进行监控。代码：https://github.com/libinYL/JpcapDemo
  • 图的深度优先遍历算法实现，用 C++ 编写，博客链接：http://libinyl.com/2018/01/22/note-graph-dfs/
