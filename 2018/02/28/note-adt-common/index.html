T(n)与O(n)

名称 含义
n 问题规模
T(n) 时间复杂度
f(n) 基本运算的频度
O(n) 时间复杂度的数量级

三者的关系：T(n)=O(f(n))

不等式

T(n)<=cO(n) c为常数
