C++概览

main函数

  • C++要求以int main()开头。
  • int main()<=>int main(void)
  • 避免使用void main()
  • 如果到达main()函数末尾没有遇到返回语句，默认认为以return 0;结尾

头文件名

  • C语言头文件：扩展名为.h
  • C++头文件：1）保留C头文件 2）纯C++头文件没有扩展名，通过namespace std使用 3）有些C头文件被转换为C++头文件，加上前缀c，如math.h->cmath，可能包含新的特性

命名空间

  • 偷懒做法：using naespace std 大型项目会有潜在问题
  • 推荐做法：只使所需名称可用。using std::cout。

语句

语句的特点：以分号结尾。

声明语句 defining declaration 和变量

  • （通常）指出要存储的数据类型和对应数据的名称，即变量，变量=值可修改，如int carrots;。
  • （复杂）引用声明，reference declaration。

赋值语句

  • 赋值语句将值赋给存储单元。carrots=2;

函数

  • 被调用的函数：called function
  • 包含函数调用的函数：calling function

函数原型语句：返回值类型 函数名（参数类型） 如double sqrt(double)，一般定义在头文件中。
