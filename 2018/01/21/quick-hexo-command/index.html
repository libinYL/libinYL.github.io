hexo常用命令速查

发表于 2018-01-21 | 分类于 技·术

创建新博客

  • 在博客文件夹内运行hexo new post "test" 即可创建test.md文件
  • 用编辑器编辑保存
  • 运行: hexo d -g

修改标签

在每篇博客的编辑状态下,修改顶端内容如

tags后面有一个空格,标签与标签间以逗号隔开.

上传图片

  • 修改_config.yml文件post_asset_folder为true
  • 此后,每次运行hexo new ,title时都会在_post文件夹下生成同名文件夹.
  • 如果需要插入图片,直接插入![](图片名称)即可,无需键入文件夹名称

上传报错

原因:连接超时,等待网络更好时尝试.
