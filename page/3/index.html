安装路径问题

使用 pip install 安装时：

  • 如果有--user参数时，则安装到:\Users\libin\AppData\Roaming\Python\Python36\site-packages
  • 如果没有--user参数，则安装到C:\Users\libin\AppData\Local\Programs\Python\Python36\Lib\site-packages下。
  • 前者没有在环境变量下，且实际上并不看重用户的区分，故没有必要用--user参数。
